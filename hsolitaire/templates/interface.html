Interface

Menu

Game

This menu allows you to operate on your games. It allows you to create a new game, load a past one or to cotinue the game that you played the last time.

  • New quick game - the only option to play Hardcore Solitaire witout registering and loggin in. It creates winnable game that is not saved.
  • Continue last game - load and starts the most recently saved game or creates a new one if there are no saved games in the database.
  • New random game - creates a new game that is fully random. Only about 40% of random games are winnable.
  • New winnable game - creates a new that is winnable. Hardcore Solitaire ccomes with around 10 000 sets of cards that are known to be winnable and randomly chooses one for you.
  • Load game - allows you to load previously played game. Hardcore Solitaire automaticaly saves your games every 30 seconds, so you will find all your not finished games here.

Save game

Games are automaticaly saved in Hardcore Solitaire, but you may want to change the default game name. If yes, then simply use this menu.

Moves

This menu allows you to control the flow of the current game.

  • Apply obvious moves - checks if there are obious moves and applies them all. Very useful at the end of game when otherwise you would have to manually collect all cards.
  • Undo last move, undo 5 moves - these commands undo your last moves. History of all your moves is saved in the database together with your game, so you are always able to bring the gaem back to the start.
  • Is game winnable? - if you make a bad move then the game may no longer be winnable. To check this simply use this menu.

Help

This is where you should go if you need some help or you would like to get more information about the game.

Gameplay

You can simply drag and drop cards to move them from one stack to the another. Game is automaticaly saved every 30 seconds. To save a game you need to have an internet connection. After each move you get information about succes (or failure) in the long thick box just above the boards. This message bos will also inform you about any possible problems.
