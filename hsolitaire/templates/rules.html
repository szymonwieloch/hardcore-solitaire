Game rules

The object

The object of the game is building four stacks of cards, one for each suit, in ascending order, from ace to king.

Starting position

Hardcore Solitaire is played with a single deck of 52 cards. Four home stacks are positioned in the center. Each home stack starts with an ace. This is where you build the piles needed to win. Other stacks are positioned around them. The game begins with all cards arranged into 10 stacks.

Moves

You can't move cards between columns at random. You are limited to moving single cards. A card can be placed either on an empty stack or on another card with the same suit and with descending order (central stacks are the oposite). So you can place a 10♠ on a J♠ , but neither on a 3♠ nor on a J♥. The only exception to this rule is the central stack at the bottom of the board. You place there one and only one cards.

Hints and tricks

  • If you can drag a card from a normal stack to a home stack, do it. Is is always a good idea.
  • Only about 40% or all random games can be won. It may be a good idea to start a winnable game (select related option in menu.
  • If you make a bad move the game may no longer be winnable. You can check game status using related option in menu. Then you can undo your last moves.
  • When you don't know what to do, try finding cards that can be moved to home stacks.
  • Having one additional empty stack really makes a life easier. Check if it possible to move cards in such a way, that you get one more empty stack.
